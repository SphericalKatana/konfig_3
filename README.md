# Ассемблер УВМ (вариант 29)

## Команды
- `load const` — загрузка 23-битной константы в аккумулятор (`A=31`)
- `read addr` — чтение из памяти по 16-битному адресу (`A=121`)
- `write addr` — запись аккумулятора в память по 16-битному адресу (`A=26`)
- `rol addr` — циклический сдвиг влево аккумулятора с операндом из памяти (`A=8`)

## Формат исходника
JSON-массив объектов:
```json
[
  {"op": "load", "const": 312},
  {"op": "read", "addr": 972}
]
